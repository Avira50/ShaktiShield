# ShaktiShield - RITM Secure Reporting Portal

Official safety and confidential reporting portal for female students of Rameshwaram Institute of Technology & Management (RITM), Lucknow.

## üöÄ GitHub Repository Description
> ShaktiShield: A secure, AI-powered safety portal for RITM Lucknow students. Featuring anonymous reporting, unique tracking codes, and automated staff dashboards with GenAI incident summarization. Built with Next.js 15, Genkit, and Tailwind CSS.

## ‚ú® Key Features
- **Confidential Reporting**: Secure and anonymous reporting of incidents with dedicated college branding.
- **Case Tracking**: Track the status of your report using a unique 8-digit code (No login or registration required for students).
- **Emergency Resources**: Quick access to Lucknow emergency helpline numbers (1090 Women Power Line, 112 UP Police).
- **AI-Powered Analysis**: Staff dashboard equipped with AI-powered incident summarization and urgency detection (using Genkit and Google Gemini).
- **Privacy & Auto-Cleanup**: Resolved cases are automatically purged from the system after 7 days of inactivity to ensure maximum student privacy.

## üõ† Deployment on Render (Step-by-Step Guide)

1. **GitHub Setup**: 
   - Push this source code to your own GitHub repository.

2. **Render Dashboard**:
   - Visit [Render.com](https://render.com/) and log in.
   - Click the **"New +"** button and select **"Blueprint"**.
   - Connect your GitHub account and select the ShaktiShield repository.

3. **Environment Variables**:
   - Render will automatically detect the `render.yaml` configuration file.
   - During the setup process, you will be prompted to enter a value for `GEMINI_API_KEY`.
   - Obtain your API key from [Google AI Studio](https://aistudio.google.com/) and paste it into the configuration field.

4. **Launch**:
   - Click the **"Apply"** button.
   - Render will begin the build process, and your portal will be live within 2-3 minutes.
